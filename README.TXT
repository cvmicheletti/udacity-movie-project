Movies Project #1

Movies Project is a a static file based web page that displays my favorite movies, their rating*, run time* and the ability to view the official YouTube.com theatrical trailer.  Created by Colleen Micheletti and whom ever at Udacity that created the fresh_tomatoes.py file.

*=All New Features!!!

Installation

Pre-requisites:
Python 2.27.10+

Download the movie.zip file and extract it where you like.
Run the following command in the extracted directory to generate the static page fresh_tomatoes.html which is the main page.  If you edit the list of movies, you will need to re-run the command to regenerate the fresh_tomatoes.html file.  When you run this command, it will spawn a browser and display the html file.

> python entertainment_center.py

What's included

Within the download you'll find the directory and files:

movies/
├── movie.py
├── entertainment_center.py
├── fresh_tomatoes.py
├── README.TXT

Provided are source a python files (*.py).  The generated html file uses CSS, jQuery and Bootstrap code from their respective providers.
